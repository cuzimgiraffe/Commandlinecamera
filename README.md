Cmaki - a Pythonbased Commandline-Styled Camera view made with cv2 and customtkinter.

# How to use

Start your Camera with the "Start Camera" Button. You now should see yourself/output of the camera. 

Click on "Terminal View" to view the Camera output in askii view/"Terminal View".

Kill/Stop your cam with "Kill Camera - Stop Camera"

#Requirements

-python 3.13
-a camera recognized by cv2 (like nearly every cam)
-Working fingers xD
-vscode or any ide you like which can handle python

# Install Steps

1. Download Python on Python.org
2. Download and extract the Program
3. open it in vs code
4. Open Terminal
5. Install Requirements with "pip install -r requirements.txt"
6. run the program with "python main.py"
